# Numpy

## Description
Used for scientific computing

Often used with plotting librariy like Matplotlib

## Installation
pip install numpy

## Notes
* define an array: 
```
x_train = np.array([1.0, 2.0])
```
* array variables have a "shape" property that the length of the array
```
x_train.shape => (2,)
x_train.shape[0] => 2 
```
