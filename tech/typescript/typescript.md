# Typescript

## Common Commands: 
* Setup a typescript project using [GTS](https://github.com/google/gts)
    * creates a code formatter, linter, and code fixer for typescript projects.
    * npx gts init
        * command to compile: npm run compile
        * command to lin
* Compile to JS
    * npx tsc index.ts
* Activate watch command
    * npx tsc -w
